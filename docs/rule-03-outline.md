# ルール: 03-outline.md

スライドの構成を決めるファイル。
`02-raw-notes.md` の素材を選び、順序と深さを整理する。

## 構成

```markdown
# タイトル

## 全体の方針

- 時間配分: ...
- 表現スタイル: ...
- トピック順序: ...

---

## スライド一覧

### 1. タイトル
- 見出し: ...
- 副題: ...

### 2. 自己紹介
- ...

### {N}. {トピック名} — {スライドの役割}
- 見出し: ...
- 画像: assets/xxx.png
- テキスト: ...
```

## 各スライドに書くこと

箇条書きで記載する

| 項目 | 内容 |
|------|------|
| 見出し | スライドの h1/h2 になるテキスト |
| 画像 | 使用する画像の概要、既にある場合もパスも併記 |
| テキスト | スライドに載せる文 |
| 小物 | ロゴやアイコン、キャラクター、絵文字など |
