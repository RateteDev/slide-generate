# ルール: 02-raw-notes.md

発表素材を蓄積するファイル。
ヒアリング・調査・メモのすべてをここに集約する。
スライドに載せるかどうかは `03-outline.md` で決めるため書きすぎても問題なし。

## 構成

トピックごとに見出しを作成し、素材を並べる。

## 書き方のルール

**素材メモ**
- 話したいこと・エピソード・データを自由に書く
- スライドに載せる深さより詳しく書いてよい（取捨選択は `03-outline.md` で行う）

**ヒアリング内容**
- 発表者の言葉をそのまま残す。整えすぎない

**調査メモ（slide-research の結果）**
- 出典URLを必ず含める
- セクションを分けて追記する（既存内容は消さない）

例
```markdown
# 調査・ヒアリングメモ

## トピック
~~~
```
