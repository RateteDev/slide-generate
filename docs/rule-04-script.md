# ルール: 04-script.md

発表時に話す内容をスライドごとにまとめた台本。

## 構成

```markdown
# 台本

<!-- 発表時間の目安: XX 分 -->

---

## スライド 1: タイトル

（話す内容）

---

## スライド 2: 自己紹介

（話す内容）
```

## 書き方のルール

**文体**
- 「です・ます」調で、実際に話す言葉に近い自然な文体で書く
- 箇条書きではなく、話す流れとして書く

**間（ま）の記入**
- 聴衆への問いかけや意図的な間は `（少し間を置く）` のように記入する

**スライドタイプ別の目安**

| スライドタイプ | 内容 | 目安時間 |
|--------------|------|---------|
| `title` | 開幕の一言・挨拶 | 10〜20秒 |
| `chapter-title` | トピック導入・予告 | 20〜30秒 |
| 説明スライド | ポイント・背景の説明 | 1〜2分 |
| `image-only` | 画像を指しながらの一言 | 30秒〜1分 |
| `closing-qr` | まとめ・締め・質問呼びかけ | 30〜60秒 |

**時間配分**
- 冒頭（タイトル + 自己紹介）: 持ち時間の10%
- 本編（各トピック）: 持ち時間の70〜75%
- まとめ + クロージング: 持ち時間の15〜20%
